## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm

### Installation
```bash
# Clone the repository
git clone [repository-url]

# Navigate to project directory
cd frontend

# Install dependencies
npm install

# Start development server
npm start
```
